the world's first website

CERN

Launched by British scientist Tim Berners-Lee at the CERN laboratory in Switzerland, the historic site went live on December 20, 1990.

Key Facts About the Website

The Purpose: It was built to explain the World Wide Web project. It taught visitors how to use hypertext, create their own web pages, and search the web.

Hardware: It originally ran on Berners-Lee's NeXT workstation computer.The Appearance: The site was purely text-based with no colors, images, or graphics.

The Software: This single server birthed the basic concepts w e use today: HTML, HTTP, and URLs

Current Status: The original site is still live and fully accessible today.
